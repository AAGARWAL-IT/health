Overview

Web health applications are increasingly being used to perform tasks that are conventionally performed by general practitioners (GPs), such as those involved in promoting health, preventing disease, diagnosis, treatment, monitoring, and support for health services.Nowadays, medical checkups are very expensive and not everyone is capable to undergo such unaffordable tests, so to cope up with all of these problems this app is designed which would prove its benefits upto much extent. This web app is built using Machine Learning and Deep Learning which can predict the results of Seven Major Health issues which are very common and just by entering the required information, we can know whether a person is suffering from these mentioned diseases or not.
